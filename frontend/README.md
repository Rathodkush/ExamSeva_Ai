# ExamSeva Frontend

This is the user interface of the ExamSeva application, designed for students to access study materials and interact with a collaborative community.

## Technology Stack
- React.js (Component-based architecture)
- Vanilla CSS (Premium Design System)
- Socket.io-client (Real-time notifications)
- Axios (HTTP client for API requests)
- React Router DOM (Single Page Navigation)

## Key Features
- Premium UI Theme: High-end Indigo and Violet design using CSS variables and modern HSL scaling.
- Interactive Dashboard: Real-time notification bell with clickable navigation to specific site features.
- Dynamic Forum: Collaborative discussion board with real-time updates and an 'Ask a Question' system.
- Study Hub: Interface for categorized document management and AI results viewing.

## Setup Instructions
1. Navigate to the frontend directory.
2. Run 'npm install' to install dependencies.
3. Configure the .env file with the REACT_APP_API_URL pointing to the backend server.
4. Run 'npm start' to run the development server.
